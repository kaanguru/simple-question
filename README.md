## Usage use yarn

``` bash
# Install dependencies
yarn
# debug android
yarn dev
# debug android no hmr
yarn devn

# Preview on device
tns preview

# Build, watch for changes and run the application
tns run

# Build, watch for changes and debug the application
tns debug <platform>

# Build for production
tns build <platform> --env.production

```
